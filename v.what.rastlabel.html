DESCRIPTION

v.what.rastlabel retrieves raster values and labels from one or more given raster maps for each point stored in a given vector map. The name of the columns with the raster values and labels are respectivelyID_name-of-raster-map and name-of-raster-map.

Optionally, the user can define raster layers without labels. In that case only raster values are uploaded to a column with the name name-of-raster-map.
